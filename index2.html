Step 4: Your task. Create and Populate JustLee Database Tables

You have seen how I handle the PostgresDemo.sql script above. For you to practice, I would like you to create and populate the JustLee database tables.

Create a new Database named JustLee, then use the JustLee Postgres SQL script to populate the database

The script called Postgres_JLDB_Build.sql is available at: Click here to download Postgres_JLDB_Build.sql

Once you are download the file to your hard drive as shown before, Launch pgAdmin, create the JustLee database, then open up the Query Tool and enter:

SELECT table_name FROM information_schema.tables WHERE table_type='BASE TABLE' AND table_schema='public';

Check if you have a table called Publisher.
